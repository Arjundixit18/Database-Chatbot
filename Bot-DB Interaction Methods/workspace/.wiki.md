# Project Summary
The Database Chatbot Project provides an AI-driven interface for querying relational databases using natural language processing. It enables users to ask complex questions and receive accurate SQL query results through a user-friendly Streamlit application. The project emphasizes the integration of AI in database management, showcasing various architectural patterns for efficient, reliable, and scalable query handling.

# Project Module Description
The project consists of several functional modules, each implementing a unique approach to database querying:
1. **Simple Prompt Agent** - Generates SQL queries directly from user questions.
2. **ReAct Agent** - Utilizes reasoning and acting for iterative query refinement.
3. **Multi-Agent System** - Employs multiple specialized agents for analysis, planning, and execution.
4. **AI Search Agent** - Leverages semantic search and context-aware query generation.

# File Description Inventory
- **ai_search_agent.py**: Implements the AI Search optimized agent for context-aware SQL generation.
- **app.py**: The main application file that runs the Streamlit interface.
- **database_manager.py**: Manages database connections, schema setup, and query execution.
- **multi_agent.py**: Implements the multi-agent architecture for complex query processing.
- **react_agent.py**: Implements the ReAct agent for iterative reasoning and query refinement.
- **requirements.txt**: Lists the necessary Python packages for the project, now updated with pinned versions to prevent dependency conflicts.
- **simple_prompt_agent.py**: Implements the simple prompt approach for generating SQL queries.
- **template_config.json**: Configuration for template management.
- **todo.md**: Outlines the project's goals and tasks to be completed.
- **utils.py**: Contains utility functions for formatting responses and handling SQL operations.

# Technology Stack
- **Python**: Main programming language.
- **Streamlit**: Framework for building the web application interface.
- **SQLite**: Lightweight database engine for data storage.
- **Pandas**: Data manipulation and analysis library.
- **Plotly**: Library for creating interactive visualizations.

# Usage
1. **Install Dependencies**: Run `pip install -r requirements.txt` to install required packages.
2. **Set Up Database**: The application automatically initializes the SQLite database with sample data.
3. **Run Application**: Execute `python -m streamlit run app.py` to start the application.
